wfETC, by Shane Calimlim, January 2014, Public Domain

This library provides functions to decode ETC1 encoded images.  It is designed
to be fast, lightweight, and cross-platform.

We use this to emulate mobile-formatted textures on PC (with Google's ANGLE),
but I'm sure there are other use cases out there.

This software is public domain.  There is no warranty, and you are free to do
what you want with it.
